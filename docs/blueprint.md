# **App Name**: ScriptaNova

## Core Features:

- AI-Powered Plot Generation: Suggest plot ideas based on user input to help overcome writer's block.
- AI Co-writing: Enable collaborative content generation using AI to help writers improve their text.
- Automated Cover Generator: Automatically create book covers for all book formats including eBook, Paperback and Hardcover.
- Landing Page: Create landing pages that highlight the benefits of using ScriptaNova. 
- Blog: Share tutorials and thought leadership about using AI to assist writing.
- Waitlist Management: Enable waitlist functionality for new product and feature releases via Firebase.
- Contact Form: Enable potential users to contact support to have their questions answered. The information should be stored in the Firestore database.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5). Evokes innovation and intelligence. Chosen for its association with AI.
- Background color: Very light gray (#F8FAFC). Almost white, promoting a focus on the content.
- Accent color: Teal (#10B981). It enhances interaction and aligns with a modern style.
- Headers: 'Space Grotesk' (sans-serif), and body text: 'Inter' (sans-serif) for a clean, modern look.
- Use a set of consistent, modern icons that complement the overall design. Icons should be simple and easily recognizable.
- Incorporate a grid-based layout for consistent and responsive design across all devices.
- Add subtle micro-animations to improve user experience, such as hover effects, loading animations, and transitions.
